# To-Do List CLI 📝 | Python + Click

🚀 A simple and efficient Command Line To-Do List Manager built using Python and Click. Easily add, list, complete, and remove tasks—all from your terminal!

🔹 Features:
✅ Add tasks with a single command
✅ List all tasks with completion status
✅ Mark tasks as completed
✅ Remove tasks when done
✅ Saves tasks using JSON storage
✅ Lightweight & easy to use
